\doxysection*{Build}

{\bfseries{Custom\+Char}} depends on SDL2 library to capture audio from the microphone. Install SDL2 library before building.


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Install SDL2 on Linux}
\DoxyCodeLine{sudo apt-\/get install libsdl2-\/dev}
\DoxyCodeLine{}
\DoxyCodeLine{\# Install SDL2 on Mac OS}
\DoxyCodeLine{brew install sdl2}

\end{DoxyCode}


Build the {\bfseries{Custom\+Char}} executable\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir build}
\DoxyCodeLine{cd build}
\DoxyCodeLine{cmake ..}
\DoxyCodeLine{make}

\end{DoxyCode}
